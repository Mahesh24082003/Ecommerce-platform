# E-Commerce Platform Case Study

This is an E-Commerce Platform project with Angular frontend and Express.js backend.

## Project Structure

```
E-Commerce-Platform-casestudy_460/
├── backend/          # Express.js API server
│   ├── server.js     # Server entry point
│   ├── app.js       # Express app configuration
│   └── package.json
├── frontend/        # Angular application
│   ├── src/         # Source files
│   └── package.json
└── package-lock.json
```

## Getting Started

### Backend Setup
```bash
cd backend
npm install
npm start
```

### Frontend Setup
```bash
cd frontend
npm install
npm start
```

## Technologies Used
- **Frontend**: Angular 21
- **Backend**: Express.js 5, Sequelize, MySQL
- **Database**: MySQL
